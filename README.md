# Domus Feed
Domus FeedはデスクトップPC向けのRSSフィードリーダーアプリです。

Domus Feed is a desktop RSS reader application.


## Download
最新のインストーラーは**GitHub Releases**ページを参照の上ダウンロードしてください。

## アプリについて
### はじめに
- バックエンドはローカル環境のみで動作します
- クラウドサービスは利用していません
- ユーザーのアプリケーションデータはローカルに保存されます
